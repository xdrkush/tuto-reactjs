# 06-webapp

# C'est quoi un Build ?

Le build vous permetra de compressé et optimiser votre application pour la production, la fonctionalité est déjà intégré par REACT en tapant la commande 'npm run build'.

# Install

```
npm i
npm run start
```

# Build

```
npm run build
```

# Run

```
sudo npm i -g serve
serve build/
```
